# 📖 독서 목록
## 개요
- 독서를 통해 교양과 창작 소재를 얻기 위함
- 만화책이어도 단순 오락용이 아닌 교양, 철학, 자기개발 등 유익하거나 영감을 줄 수 있는 책도 독서 대상에 포함.   
- 완독에 가까운 90 % 이상 읽은 책도 포함.
- 주로 책의 일부만 참고하는 기법서나 교재 같은 경우 제외
